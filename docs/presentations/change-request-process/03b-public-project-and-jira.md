# Step 1 (cont.): Public Project & JIRA Mirroring

## Public Project - Internal & External Contributors

This is a **public open-source project**. Tickets are raised by both **internal and external teams** directly on GitHub. GitHub Issues are the **single source of truth** for each component's status.

## Optional: JIRA Mirroring

- A JIRA ticket can optionally be auto-raised to mirror a GitHub Issue for internal tracking
- As a public project, JIRA may not accurately reflect the current status of a component
- External contributions, comments, and review activity happen on GitHub

**GitHub remains the authoritative record.**

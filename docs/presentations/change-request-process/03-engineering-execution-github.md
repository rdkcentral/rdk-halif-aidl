# Engineering Execution - GitHub

This is a **public open-source project**. GitHub provides a **complete, auditable git history** of every change - visible to all contributors, internal and external.

- All change context lives **where every contributor can see it**
- Without this, the repository is littered with internal references that external teams cannot access
- External and internal contributors work from the **same source of truth**
- Engineering status is **synced back to JIRA** based on PM planning requirements
- **RAG status visible to all parties** - component review progress is published openly

See: [Component RAG Status](https://rdkcentral.github.io/rdk-halif-aidl/) | [Engineering-Oriented Goal Setting](https://rdkcentral.github.io/rdk-halif-aidl/0.13.0/whitepapers/engineering_goals/#engineering-oriented-goal-setting) | [Versioning & Review SOP](../../governance/versioning-sop.md)

**GitHub owns the engineering execution. Engineering truth lives in GitHub.**

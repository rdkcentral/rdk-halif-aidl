# Step 6: Merge & Component Lifecycle

After successful review, the component progresses through a **RAG status lifecycle**:

| Status | Name                    | Meaning                                           |
|--------|-------------------------|-------------------------------------------------|
| 🔴     | **RED - Not Ready**     | Strategy and requirements gathering phase        |
| 🟡     | **AMBER - Under Review**| AIDL files exist, in or awaiting 14+5 review    |
| 🟢     | **GREEN - Approved**    | All reviewers signed off, ready for implementation|

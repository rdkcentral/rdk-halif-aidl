# Step 6: Merge & Component Lifecycle

After successful review, the component progresses through a **RAG status lifecycle**:

## 🔴 RED - Not Ready

- Strategy and requirements gathering phase
- Interface files may not yet exist

## 🟡 AMBER - Under Active Review

- AIDL interface files exist and are feature-complete
- Currently in or awaiting the 14+5 review cycle

## 🟢 GREEN - Reviewed & Approved

- All mandatory reviewers signed off
- Stable on develop branch
- **Ready for vendor implementation**

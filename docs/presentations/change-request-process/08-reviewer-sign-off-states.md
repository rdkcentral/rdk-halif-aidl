# Step 5: Who Reviews - Sign-Off States

Each reviewer records a formal status against the PR:

| Status               | Symbol | Meaning                                    |
|----------------------|--------|--------------------------------------------|
| **Pending**          | ☐      | Review not yet started                     |
| **In Review**        | 🔍     | Actively reviewing during 14-day window    |
| **Changes Requested**| 🔁     | Feedback given, awaiting resolution        |
| **Recheck**          | 🔄     | Previously reviewed, needs re-verification |
| **Reviewed**         | ✅     | Signed off and approved                    |
| **Abstained**        | ➖     | Not applicable for this component          |

**All mandatory reviewers must reach "Reviewed" or "Abstained" before merge.**
